## Features NOT Documented Anywhere

These features exist in code but have no documentation entry:

1. **Stdlib filter in RWR** (context.go:539): nodes prefixed with `stdlib://` are filtered from RWR results alongside `external://` nodes. Not mentioned in noise filtering docs.
2. **IndexFilesIncremental**: incremental file-scoped reindex (used by daemon for changed files). Exists in bench but not documented as a feature.
3. **Adjacency cache v2 format**: the 65 bytes/edge compact binary format. Documented in retrieval-pipeline.md lines 377-381 but not in features.md.
4. **codegraph benchmark adapter** (`bench/cross-system/adapters/codegraph.go`): allows head-to-head comparison. No feature entry.
5. **Query latency 2ms result**: exists in FINDINGS.md and roadmap but not in any architecture doc as a "current state" claim. The retrieval-pipeline.md doesn't state it.
